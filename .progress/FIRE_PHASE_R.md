## 什么是 Phase R？

Phase R (Review) 是在每个阶段完成后执行的全面审查流程，用于：

1. **评估当前状态** - 对比计划和实际完成度
2. **识别问题** - 发现技术债务和架构差距
3. **规划下一步** - 基于审查结果制定新计划
4. **持续改进** - 通过反馈循环优化流程

---

## 🔥 立即启动 Phase R

在 INBOX.md 中添加：

```
🔥 启动 Phase R 审查
```

或者直接使用命令：

```markdown
@workspace 执行 Phase R 审查：
1. 全面审查项目状态
2. 生成审查报告
3. 检查 INBOX 新指令
4. 更新下阶段计划
5. 继续自主推进
```

---

## Phase R 输出

1. **审查报告** → `reports/AUDIT_REPORT_XXX.md`
2. **新计划** → `plans/plan_XXX.json`
3. **状态更新** → `PROGRESS_STATUS.json`
4. **任务清单** → `TODOS.md`
